# Эволюция рабочего окружения

Я долгое время был фанатом Linux. Я пробовал **Arch**, **CachyOS**, **Pop!_OS**. Я часами настраивал **Hyprland** и **Quickshell**, вылизывая каждый пиксель интерфейса. Это было круто, я чувствовал себя настоящим хакером.

## Почему Windows + WSL2?

В какой-то момент я понял, что трачу больше времени на *настройку* системы, чем на *работу*. Последней каплей стала бесконечная возня с настройкой конфигурации для работы с моей дискретной видеокартой **NVIDIA**. Кто пробовал завести Wayland на Nvidia, тот знает, через какую боль приходится пройти.

> Linux is free if your time is free.

Windows 11 с подсистемой **WSL2** (Ubuntu) дает мне лучшее из двух миров:
1. **Windows:** Стабильный GUI, беспроблемная работа видеокарты, Photoshop/DaVinci Resolve для хобби.
2. **WSL2:** Полноценный Linux-терминал для разработки (Node.js, Docker, Git).

## Мой стек инструментов

- **Editor:** VS Code.
- **Terminal:** Windows Terminal + Bash.
- **Notes:** Obsidian (синхронизация через Git).
- **Browsers:** Chrome, Firefox, Zen, Comet (Perplexity).

Теперь я просто работаю, и это прекрасно.
